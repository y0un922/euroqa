# 模块: frontend.components.Sidebar

## 职责

- 展示左侧历史会话、文档导航、上传入口和热门问题
- 提供“新建检索会话”按钮，用于归档当前会话并切换到新的空白会话
- 提供历史会话入口，点击后可切换显示已归档的本地会话

## 行为规范

- 侧边栏不再展示“术语预览”，该区域由“历史会话”取代。
- “新建检索会话”会把当前非空会话归档到历史会话列表，再打开新的空白会话。
- 历史会话列表为空时，展示归档提示文案；有历史时按最新归档顺序显示标题、问答条数和更新时间。
- 热门问题继续来自 `/api/v1/suggest`，组件只截取前 6 条显示，避免左栏过长。
- 文档列表继续负责高亮当前选中文档，并在处理中的文档上展示状态与进度。

## 依赖关系

- 依赖 `frontend/src/components/DocumentUpload.tsx` 提供文档上传入口
- 依赖 `frontend/src/components/DocumentStatusBadge.tsx` 展示文档处理状态
- 依赖 `frontend/src/hooks/useEuroQaDemo.ts` 提供当前会话摘要、历史会话切换和新建会话回调
